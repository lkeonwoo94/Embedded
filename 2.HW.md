## 1. MCU-CPU-Core 차이
Core = Register Bank + ALU + Barrel Shifter + Multiplier 
Processor(CPU) = Core + Co-processor, MMU, Cache Memory
MCU = CPU + Memory-Controller + GPIO , LCD, Sound ...
